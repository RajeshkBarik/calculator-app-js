A simple and responsive calculator web application built using **HTML, CSS, and JavaScript**.

## 🚀 Features
- Basic arithmetic operations (+, -, *, /)
- Responsive UI design
- Clean and user-friendly interface
- Real-time calculation display

## 🛠️ Tech Stack

- CSS3
- JavaScript (ES6)
